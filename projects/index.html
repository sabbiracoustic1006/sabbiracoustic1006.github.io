eBay 2024 University ML Competition

Vehicle Parts Compatibility Extraction - 2nd Runner-up

Developed a machine learning solution to extract vehicle parts compatibility (fitment) data from eBay listings:

  • Data Preparation: Converted raw HTML data to text and cleaned text using Qwen2.5-32B-Instruct for noise reduction and irrelevant information removal
  • Model Development: Fine-tuned (k-fold) two open-source LLMs (Mistral-Nemo and Qwen2.5-14B) using LoRA on labeled training data
  • Ensemble Approach: Implemented majority voting strategy to combine predictions from k-fold models
  • Results: Achieved 79.6% F0.2 score, securing second runner-up position in the competition

eBay 2023 University ML Competition

Named Entity Recognition - 2nd Runner-up

Built a model to extract and label named entities from eBay item titles:

  • Teacher-Student Framework: Implemented a teacher-student training approach with XLM-ROBERTA and ELECTRA models
  • Semi-supervised Learning: Generated pseudo labels on unlabeled data using the teacher model while filtering uncertain samples
  • Advanced Training: Used k-fold training and ensemble strategies to combine predictions from the k-fold models
  • Results: Achieved an F1 score of 94.37%, securing second runner-up position in the challenge
